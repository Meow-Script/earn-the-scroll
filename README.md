# Earn the Scroll

An iOS app built with Swift and Xcode designed to help users reduce mindless scrolling by making screen time something you earn instead of something you default to.

---

## Overview

Earn the Scroll is a productivity and habit-building app that encourages intentional phone usage. Instead of endlessly scrolling, users complete tasks or goals to unlock screen time.

---

## Features

* Track and limit screen time
* Complete tasks to unlock usage
* Clean and simple iOS interface
* Encourages healthier digital habits

---

## How to Run

1. Clone the repository:

```
git clone https://github.com/Meow-Script/earn-the-scroll.git
```

2. Open the `.xcodeproj` file in Xcode
3. Build and run on a simulator or device
